# Credit Scoring Model

This project predicts whether a loan applicant is creditworthy using machine learning.

## Features Used
- Age
- Gender
- Education
- Income
- Debt
- Credit Score
- Loan Amount
- Loan Term
- Number of Credit Cards
- Payment History
- Employment Status
- Residence Type
- Marital Status

## Model
Random Forest Classifier

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score
- ROC-AUC

## Files
- credit_model.py → Model training script
- test_model.py → Desktop GUI application
- credit_model.pkl → Trained model
- scaler.pkl → Saved feature scaler
- data.csv → Dataset

## How to Run

### Train the Model
python credit_model.py

### Run the GUI Application
python test_model.py
